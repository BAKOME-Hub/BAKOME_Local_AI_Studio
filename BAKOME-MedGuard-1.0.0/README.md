## 📖 Description

**EN** — BAKOME-MedGuard is a production-ready AI medical diagnostic engine written in pure Rust. It uses weighted symptom matching to screen 15 diseases, recommends radiology imaging (XRAY, CT, MRI, Ultrasound), detects 8 viruses, and provides an interactive web dashboard with real-time statistics. Built with Axum web framework and zero-cost open source licensing.

**ES** — BAKOME-MedGuard es un motor de diagnóstico médico con IA listo para producción escrito en Rust puro. Utiliza coincidencia ponderada de síntomas para detectar 15 enfermedades, recomienda estudios de imagen (rayos X, TC, resonancia magnética, ecografía), detecta 8 virus y proporciona un panel web interactivo con estadísticas en tiempo real. Construido con el framework web Axum y licencia de código abierto sin costo.

**ZH** — BAKOME-MedGuard 是一个用纯Rust编写的生产级AI医疗诊断引擎。它使用加权症状匹配筛查15种疾病，推荐放射影像（X光、CT、MRI、超声），检测8种病毒，并提供实时统计的交互式网页仪表板。使用Axum Web框架构建，零成本开源许可。

## ⚡ Features

| Module | Description |
|--------|-------------|
| 🧠 **Symptom Diagnosis** | 15 diseases with weighted confidence scoring |

## ⚙️ Quick Install

```bash
git clone https://github.com/BAKOME-Hub/BAKOME-MedGuard.git
cd BAKOME-MedGuard
cargo build --release
cargo run --release
```

Open http://localhost:3001 in your browser.
